🚀 **Secure 2FA v0.1.0 - 첫 공식 데스크톱 릴리즈**

## 💡 Secure 2FA란?
**Secure 2FA**는 외부 클라우드 통신을 완전히 배제하고, 오프직 사용자의 로컬 환경 컴퓨터 내에서만 안전하게 동작하도록 설계된 **오프라인 전용 오픈소스 보안 인증기(OTP)** 입니다. 

모든 시크릿 키와 인증 메타데이터는 오직 사용자가 지정한 '마스터 PIN'으로만 보호되며, Rust 언어 기반의 강력한 암호화 알고리즘 처리를 바탕으로 보안 스토리지(SQLite)에 저장됩니다.

---

## ✨ 0.1.0 주요 업데이트 기능 (Release Highlights)

### 1. 강력한 PIN 기반 오프라인 보안
- **로컬 암호화**: 첫 실행 시 4~6자리의 마스터 PIN을 설정합니다. 시스템은 Rust의 `ring::pbkdf2` 시스템을 사용해 솔트(Salt)와 반복 연산을 거쳐 PIN을 단방향 암호화합니다.
- **실시간 잠금**: 상단 로고(방패 모양)를 클릭하거나, 앱 세션이 만료/새로고침될 시 즉각적으로 전체 데이터 화면에 접근이 차단되는 잠금 화면 모드를 지원합니다. 인증 성공 전까지 내부 스토리지 데이터 접근은 완벽히 차단됩니다.

### 2. 사용자 친화적 모던 UI (Glassmorphism)
- **커스텀 타이틀바**: 거추장스러운 기본 운영체제 창 테두리를 제거하고, 앱 테마와 어우러지는 글래스모피즘(투명 유리 효과) 테마를 탑재했습니다. 완전히 커스텀된 창 버튼(최소/최대/닫기)을 제공합니다.
- **Sticky 네비게이션**: 인증 계정이 수십 개로 늘어나더라도 항상 가장 상단에 고정되어 있는 헤더 메뉴를 통해 계정 추가나 셋업, 백업을 쉽게 수행할 수 있습니다.
- **계정 관리 FAB**: 접근성이 높은 화면 우측 하단의 플로팅(+) 액션 버튼을 통해 어디서든 새로운 2FA 계정을 입력할 수 있습니다.

### 3. 간편한 백업 & 복원 기능
- **단일 JSON 내보내기/불러오기**: 등록된 여러 개의 2FA 스토리지 데이터를 단 한 번의 클릭으로 .json 백업 파일로 추출할 수 있습니다. 추출된 백업은 언제든 앱 화면에 드래그&드롭(Drag & Drop)하여 곧바로 복구할 수 있습니다.

### 4. 빠르고 정확한 실시간 OTP 카운트다운
- RFC 6238 스펙을 철저히 준수한 TOTP 코드를 지원하여 Discord, GitHub 등 주요 인증 서버의 토큰을 딜레이 없이 제공합니다.
- 코드를 한 번만 클릭하면 클립보드에 자동 복사되어 원활한 워크플로우를 보장합니다.
